# lerolero.com
A remake of LeroLero.com built in AngularJS

Make sure you have grunt cli installed globablly on your system, otherwise install it running:
```
npm install -g grunt
```

Install project dependencies:
```
bower install
npm install
```

Run the server with:

```
grunt server --force
```
